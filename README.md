# ulisapp
イベント参加者管理用アプリ

「大まかな仕様」

一般部員用と幹部用とで別々の認証を行う

    一般部員用ページ
        ・イベント一覧の閲覧
        ・イベント詳細の閲覧
        ・イベント参加者の編集、削除
    幹部用ページ(Admin)
        ・イベント一覧の閲覧
        ・イベント詳細の閲覧
        ・イベント参加者の編集、削除
        ・各イベントの編集、削除

「その他」
    サーバ環境変数で
        NORMAL_USER
        NORMAL_PASS
        ADMIN_USER
        ADMIN_PASS
    を設定する。
